# assets 文件夹

存放类似下拉菜单选项json文件

```json
[
    {
        'name': 'name1',
        'value': '900'
    }
];
```

### 1. data 文件夹

存放数据

文件命名方式:驼峰式

##### 1.1. cascader.json

级联菜单列表

##### 1.2. menuList.json

导航菜单列表


### 2. images 图片

##### 2.1. 公用

类型_功能 | 类型_功能_标记

如 `icon_logo` | `icon_password` | `icon_password_diabled`

##### 2.2. 私有

模块_类型_标记 |模块_类型_功能 | 模块_类型_功能_标记

如 `login_bg_1` |`login_icon_password` | `login_icon_password_diabled`
